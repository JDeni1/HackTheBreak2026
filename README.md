# HackTheBreak2026

## Our Project

Shortcuts, a website for beginners with no experiance in coding to learn and explore new enviorments!

The intimidation of code is hard enough, why make it harder by the information overload of going through
countless documents and set up videos to have a working IDE?

Using Shortcuts, you not only get a library of shortcuts commonly used within major Industry Standard IDEs, we also provide detailed "How-to" guides ranging from the creation of your first project to opening a new class.

TO DEPLOY : Type "fly deploy" in terminal
